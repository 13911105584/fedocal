API

Fedocal provides a small API to retrieve information from the database in JSON.

Retrieve meetings in a time frame

In order to retrieve all the meetings within a given time frame you will need to query {{ url_for('api_date', calendar='calendar_name', start_date='start_date', end_date='end_date') }}

The arguments are:

    • calendar: the name of the calendar.
      start_date: the start date of the period to consider. Format would be of type yyyy-mm-dd.
      end_date: the end date of the period to consider. Format would be of type yyyy-mm-dd.

    • The last two arguments (start_date &s; end_date) are actually optionnals meaning that you may call {{ url_for('api_date_default', calendar='calendar_name') }}. This will by default return you the coming six months.

    Retrieve meeting in a time frame in a region

    In order to retrieve all the meetings within a given time frame and for a specific region, you will need to query {{ url_for('api_place', region='region', calendar='calendar_name', start_date='start_date', end_date='end_date') }}

    The arguments are:

    • region: the name of the region you would like to have the events off.
      calendar: the name of the calendar.
      start_date: the start date of the period to consider. Format would be of type yyyy-mm-dd.
      end_date: the end date of the period to consider. Format would be of type yyyy-mm-dd.

    • The last two arguments (start_date &s; end_date) are actually optionnals meaning that you may call {{ url_for('api_place_default', region='region', calendar='calendar_name') }}. This will by default return you the coming six months.
